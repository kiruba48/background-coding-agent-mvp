    Class structure:
    ```
    export class ClaudeCodeSession {
      private config: SessionConfig
      private abortController: AbortController | null = null

      constructor(config: SessionConfig)
      async start(): Promise<void> {}  // no-op
      async run(userMessage: string, logger?: pino.Logger): Promise<SessionResult>
      async stop(): Promise<void>  // abort via AbortController
    }
    ```

    Internal helpers (module-level or private):
    - `buildPreToolUseHook(workspaceDir: string, logger: pino.Logger)` -- returns HookCallback
      - Resolve workspaceDir to absolute path
      - SENSITIVE_PATTERNS: /^\.env$/, /^\.env\./, /^\.git\//, /\/\.git\//, /private_key/i, /\.pem$/, /\.key$/
      - Check toolInput.file_path ?? toolInput.path
      - If no path (Bash tool): return {} (allow)
      - If path outside repo: return { systemMessage, hookSpecificOutput: { hookEventName: 'PreToolUse', permissionDecision: 'deny', permissionDecisionReason } }
      - If path matches sensitive pattern: same deny structure
      - Log blocked attempts: logger.warn({ type: 'audit', tool, path, reason, toolUseId }, 'tool_blocked')
      - Otherwise return {} (allow)

    - `buildPostToolUseHook(logger: pino.Logger, counterRef: { count: number })` -- returns HookCallback
      - Increment counterRef.count
      - Extract file_path ?? path from toolInput
      - logger.info({ type: 'audit', tool: postInput.tool_name, path, timestamp: new Date().toISOString(), toolUseId }, 'file_changed')
      - Return {}

    - `mapSDKResult(finalResult, sessionId, toolCallCount, duration, logger)` -- returns SessionResult
      - If no finalResult: return failed with 'No result message received'
      - Log SDK cost data: logger.info({ totalCostUsd, numTurns, usage }, 'sdk_session_cost')
      - switch on finalResult.subtype:
        - 'success' -> status: 'success', finalResponse: finalResult.result
        - 'error_max_turns' -> status: 'turn_limit', error: 'Turn limit exceeded'
        - 'error_max_budget_usd' -> status: 'turn_limit', error: 'Session budget exceeded'
        - 'error_during_execution' / default -> status: 'failed', error from finalResult.errors or 'Session failed'

    run() method:
    - Generate sessionId via crypto.randomUUID()
    - Create AbortController, store as this.abortController
    - Build hooks
    - Call query({ prompt: userMessage, options: { cwd: workspaceDir, maxTurns, maxBudgetUsd: 2.00, permissionMode: 'acceptEdits', disallowedTools: ['WebSearch', 'WebFetch'], model: config.model, abortController, hooks: { PreToolUse: [{ matcher: 'Write|Edit', hooks: [preHook] }], PostToolUse: [{ matcher: 'Write|Edit', hooks: [postHook] }] }, settingSources: [] } })
    - for await (const message of queryGen) { if message.type === 'result' then capture }
    - In finally: call queryGen.return(undefined) wrapped in try/catch, null out abortController
    - Return mapSDKResult(...)
