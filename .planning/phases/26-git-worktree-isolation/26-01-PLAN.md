    **Static method `buildWorktreePath(repoDir: string, suffix: string): string`:**
    - `const parentDir = path.dirname(repoDir);`
    - `const repoBasename = path.basename(repoDir);`
    - Return `path.join(parentDir, '.bg-agent-' + repoBasename + '-' + suffix)`

    **Constructor:** `constructor(private repoDir: string, private worktreePath: string, private branchName: string)`

    **Method `async create(): Promise<void>`:**
    - `await execFileAsync('git', ['worktree', 'add', this.worktreePath, '-b', this.branchName], { cwd: this.repoDir });`
    - Write PID sentinel as JSON: `await writeFile(path.join(this.worktreePath, '.bg-agent-pid'), JSON.stringify({ pid: process.pid, branch: this.branchName }));`

    **Method `async remove(): Promise<void>`:**
    - Try: `await execFileAsync('git', ['worktree', 'remove', '--force', this.worktreePath], { cwd: this.repoDir });`
    - Catch: log warning but do not rethrow (best-effort in finally block)
    - Try: `await execFileAsync('git', ['branch', '-D', this.branchName], { cwd: this.repoDir });`
    - Catch: log warning but do not rethrow (branch may already be deleted)

    **Static method `async pruneOrphans(repoDir: string, logger?: pino.Logger): Promise<void>`:**
    - `const parentDir = path.dirname(repoDir);`
    - `const repoBasename = path.basename(repoDir);`
    - `const prefix = '.bg-agent-' + repoBasename + '-';`
    - Read entries from `parentDir` via `readdir()`. If readdir fails, return silently.
    - For each entry starting with `prefix`:
      - `const worktreePath = path.join(parentDir, entry);`
      - Check it's a directory via `stat()`. If not, skip.
      - Read `.bg-agent-pid` sentinel file. Parse as JSON `{ pid: number, branch: string }`.
      - If sentinel missing, unparseable, or pid is NaN: mark as orphan.
      - If pid present: try `process.kill(pid, 0)`. If throws with code ESRCH: orphan. If throws with EPERM: alive (treat as non-orphan). If no throw: alive.
      - If orphan:
        - `logger?.warn({ worktreePath }, 'Pruning stale worktree');`
        - Try `execFileAsync('git', ['worktree', 'remove', '--force', worktreePath], { cwd: repoDir })` (catch and ignore)
        - If sentinel had a valid branch name: try `execFileAsync('git', ['branch', '-D', branch], { cwd: repoDir })` (catch and ignore)

    **Getter properties:** `get path(): string` returns `this.worktreePath`, `get branch(): string` returns `this.branchName`.
