    - Returns fallback message when no recognizable output

    npmBuildVerifier:
    - Skips (passed:true, durationMs:0) when no package.json exists

    - Returns passed:false with error summary on non-zero exit

    - Uses error type 'build' (same as TS build -- flows through existing retry loop)

    npmTestVerifier:
    - Skips (passed:true, durationMs:0) when no package.json exists

    - Returns passed:false with error summary on non-zero exit
    - Handles timeout (300s)
    - Uses error type 'test' (same as vitest -- flows through existing retry loop)

    compositeVerifier integration:
    - npm verifiers run sequentially after Maven verifiers (npm test may invoke npm build internally)
    - Skip npm test if npm build failed (same pattern as Maven: noise reduction)
    - Error ordering: Build > Test > Maven Build > Maven Test > npm Build > npm Test > Lint
    - All existing composite tests still pass
  </behavior>
