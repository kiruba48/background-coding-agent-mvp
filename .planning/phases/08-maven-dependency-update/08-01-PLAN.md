  <action>
    In `src/cli/index.ts`, add two new `.option()` calls (NOT `.requiredOption()` -- these are conditionally required):

    ```
    .option('--dep <groupId:artifactId>', 'Dependency to update (e.g., org.springframework:spring-core)')
    .option('--target-version <version>', 'Target version for dependency update')
    ```

    In the `.action()` handler, AFTER existing validation blocks and BEFORE the `runAgent()` call, add conditional validation:

    ```typescript
    const depRequiringTaskTypes = ['maven-dependency-update'];
    if (depRequiringTaskTypes.includes(options.taskType)) {
      if (!options.dep) {
        console.error(pc.red('Error: --dep is required for task type: ' + options.taskType));
        process.exit(2);
      }
      if (!options.targetVersion) {
        console.error(pc.red('Error: --target-version is required for task type: ' + options.taskType));
        process.exit(2);
      }
    }
    ```

    Pass the new fields to `runAgent()`:
    ```typescript
    dep: options.dep as string | undefined,
    targetVersion: options.targetVersion as string | undefined,
    ```

    Update RunOptions in `src/cli/commands/run.ts` to add:
    ```typescript
    dep?: string;            // groupId:artifactId for dependency update tasks
    targetVersion?: string;  // target version for dependency update tasks
    ```

    Important: The `depRequiringTaskTypes` array is designed to be extended later by Phase 9 (add 'npm-dependency-update').
  </action>
