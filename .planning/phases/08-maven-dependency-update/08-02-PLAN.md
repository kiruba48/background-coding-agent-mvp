<objective>
Add Maven build-system detection to the composite verifier so Maven projects get build and test verification.

Purpose: When the agent updates a pom.xml dependency, the verifier detects Maven and runs `mvn compile` + `mvn test` to catch build failures and test regressions (MVN-03). Failed verification feeds errors through the existing retry loop, enabling the agent to fix breaking API changes (MVN-04).
Output: mavenBuildVerifier, mavenTestVerifier in verifier.ts; Maven error summarizers in summarizer.ts; updated compositeVerifier running Maven alongside TypeScript verifiers.
</objective>
